**Chess Clock**

author: Khanh Nguyen

Framework: Angular 12

A simple chess clock with the most common presets.

The clock can also be controlled by keyboard input:
  - `1` - Clock 1 control
  - `2` - Clock 2 control
  - `P` or `Spacebar` - Pause both clocks
  - `R` - Reset

See working version here: https://chessclock.us
